# Linux Server Documentation

This PDF documents the setup of an Ubuntu Linux server running in Hyper-V. It focuses on adding and configuring a new virtual disk, creating an ext4 filesystem, mounting the disk to `/mnt/data`, and making the mount persistent through `/etc/fstab`.

The documentation also covers creating users and groups, setting ownership and permissions for directories `A` and `B`, testing access rights with different users, working with symbolic and hard links, and using the `find` command to search files by owner, group, and permissions.

The goal of the project was to demonstrate basic Linux server administration, disk management, user/group permissions, and filesystem operations.
