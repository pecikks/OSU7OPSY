# Windows Server Documentation

This PDF documents the setup of a Windows Server environment in Hyper-V. The server was configured as a domain controller with Active Directory, DNS, file sharing services, SMB/NTFS permissions, FSRM quotas, and Group Policy drive mapping.

The documentation covers the creation of the `FIRMA.local` domain, organizational units, security groups, user import from CSV, shared folders for departments and home directories, access permissions, quota configuration, and functional testing with a domain user.

The goal of the project was to demonstrate a working Windows Server infrastructure for a small company with separate departments and controlled access to shared resources.
